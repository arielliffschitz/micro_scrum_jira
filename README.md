# micro_scrum_jira

A small Spring Boot microservice project for managing a Scrum backlog, built with Java 21 and Spring 3.5.8.  
This project is a professional practice exercise to demonstrate backend skills, JPA usage, REST endpoints, and clean coding principles.  

## Features

- User management (CRUD)  
- Product backlog management (CRUD)  
- Sprint backlog creation from product backlog  
- Task assignment and tracking  
- Basic authentication and role-based access control  

## Tech Stack

- Java 21  
- Spring Boot 3.5.8  
- Spring Data JPA  
- MySQL  
- Maven  

## Getting Started

1. Clone the repository.  
2. Create a MySQL database and update `application.properties` with your credentials.  
3. Run the application with your IDE or `mvn spring-boot:run`.  

## Notes

This project is a work-in-progress and serves as a demonstration of professional backend development practices.
